## 🚀 Como Usar

1. **Abrir o Site**:
   - Basta abrir o arquivo `index.html` em qualquer navegador moderno
   - Ou hospede os arquivos em um servidor web

2. **Personalização**:
   - **Informações de Contato**: Edite os dados no arquivo `index.html` nas seções de contato e footer
   - **Telefone WhatsApp**: Altere o número na linha do botão flutuante (procure por `5571999999999`)
   - **Cores**: Modifique as variáveis CSS em `:root` no arquivo `styles.css`
   - **Conteúdo**: Ajuste textos e descrições diretamente no `index.html`

3. **Imagens**:
   - As imagens atuais são do Unsplash (CDN gratuito)
   - Para usar suas próprias fotos, baixe imagens e substitua os URLs nos atributos `src`
   - Mantenha dimensões adequadas para carregamento rápido
